Scanner

Obtenable en début de jeu.

  • Type : Objet
  • Ingrédients : 1x Pile, 1x Titane (Fabricateur)

Couteau

Utile pour la récolte et la défense en cas d'attaque.

  • Type : Objet
  • Ingrédients : 1x Caoutchouc en silicone, 1x Titane (Fabricateur)

Constructeur d'habitat

Obligatoire pour la construction des bases sous-marines.

  • Type : Objet
  • Ingrédients : 1x Kit de câblage, 1x Circuit intégré, 1x Pile (Fabricateur)

Outil de réparation

Répare les structures et les véhicules abîmés ou endommagés.

  • Type : Objet
  • Ingrédients : 1x Caoutchouc en silicone, 1x Soufre de cave, 1x Titane (Fabricateur)

Fabricateur

Essentiel pour fabriquer de nouveaux items et faire cuire les aliments.

  • Type : Appareil
  • Ingrédients : 1x Titane, 1x Or, 1x Echantillon de corail plat (Constructeur d'habitat)

Radio

Dans le jeu, permet d'écouter les transmissions des survivants du crash de l'Aurora.

  • Type : Appareil
  • Ingrédients : 1x Titane, 1x Cuivre (Constructeur d'habitat)
